# Frontend - ForEachApp

### `npm install`
Realizar instalación de los modulos de nodejs

### `npm start`
Una vez instado los modulos, ejecutar la APP
Link: [http://localhost:3000](http://localhost:3000).

### Caracteristicas
- Peticiones fetch API
- Redux
- Validaciones y mensajes con Swal
- Bootstrap
